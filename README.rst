Pylama_gjslint
#############

pylama_gjslint -- Support gjslint in Pylama_.

.. contents::


Requirements
=============

- python >= 2.6


Installation
=============

**Pylama_gjslint** should be installed using pip: ::

    pip install pylama_gjslint
